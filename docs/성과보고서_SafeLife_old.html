기술요약
(3줄 이내)		 ㅇ SafeLife는 고령자의 디지털 생활 안전을 위한 AI 통합 플랫폼으로, 음성 바코드 리더, 보이스피싱 실시간 감지, 키오스크 AI 도우미, 보호자 대시보드 4가지 핵심 기능을 제공합니다.

- Claude AI, Web Speech API, ZXing 바코드 인식 등 최신 AI 기술을 활용하여 고령자가 디지털 기기를 쉽고 안전하게 사용할 수 있도록 지원하며, PWA 기술로 설치 없이 모바일/PC에서 즉시 사용 가능합니다.
최종목표
(2줄 이내)		 ㅇ 고령자의 디지털 접근성을 향상시키고, 보이스피싱 등 디지털 범죄로부터 보호하여 누구나 안전하고 편리한 디지털 생활을 영위할 수 있도록 지원

- 보호자와 실시간 연동하여 가족 모두가 안심할 수 있는 통합 AI 생활 안전 생태계 구축
개발 내용
및
결과		 ㅇ 음성 바코드 리더 (들리는 바코드)
- 바코드 스캔 시 식약처 공공데이터 API 조회하여 제품명, 제조사, 원재료, 알레르기 유발 성분, 주의사항을 음성(TTS)으로 안내
- ZXing 바코드 인식, Web Speech API, 식약처 공공데이터 연동 구현

ㅇ 보이스피싱 실시간 감지 (AI 안심 가드)
- 통화 내용을 Claude AI로 실시간 분석하여 기관 사칭, 개인정보 요구, 송금 요구 등 피싱 패턴 탐지
- 금융감독원/검찰 사칭, 대출 사기, 가족 사칭 등 10가지 유형 시나리오 탐지 가능

ㅇ AI 키오스크 도우미 (벗님)
- 키오스크 화면 인식하여 단계별 음성 안내 제공 (패스트푸드, 카페, 기차표, ATM, 병원 접수)
- 시나리오 기반 가이드로 고령자의 독립적인 키오스크 사용 지원

ㅇ 보호자 대시보드 및 PWA 앱
- 연결된 고령자의 실시간 안전 상태, 활동 기록, 보이스피싱 차단 내역 모니터링
- PWA 기술로 설치 없이 홈 화면 추가하여 네이티브 앱처럼 사용, 오프라인 지원

※ 기술 스택: Next.js 15, TypeScript, Claude AI, Web Speech API, ZXing, PocketBase, Vercel
개발
과정에서
애로점		 ㅇ 바코드 인식 정확도: 다양한 조명 환경에서 인식률 저하 문제를 ZXing TRY_HARDER 옵션과 다중 포맷 지원으로 해결

ㅇ 식약처 API 데이터 한계: 미등록 제품 조회 실패 시 PocketBase 내부 DB를 백업으로 구축하여 대응

ㅇ 보이스피싱 탐지 정확도: 정상 금융 상담과 피싱 구분을 위해 Claude AI 맥락 분석과 복합 패턴 매칭 적용
기대효과 ㅇ 사회적 효과
- 디지털 격차 해소: 고령자 73%가 겪는 디지털 기기 사용 어려움 완화
- 금융 범죄 예방: 연간 1.8조원에 달하는 보이스피싱 피해 감소 기여
- 가족 간 소통 강화: 보호자 대시보드를 통한 안심 제공

ㅇ 경제적 효과
- 의료비 절감: 알레르기 사고 예방을 통한 응급 의료 비용 감소
- 행정 비용 절감: 키오스크 사용 지원으로 대면 서비스 의존도 감소
- 고령자 자립 지원: 독립적인 생활 유지로 복지 비용 절감

ㅇ 기술적 효과
- AI 접근성 향상: 복잡한 AI 기술을 쉬운 인터페이스로 제공하는 모범 사례 구축
- 공공데이터 활용: 식약처 등 정부 공공데이터와 AI 융합 모델 제시
- PWA 기술 확산: 앱 설치 장벽 없는 서비스 배포 방식 보급
