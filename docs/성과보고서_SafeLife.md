## 기술요약 (3줄 이내)

- SafeLife는 고령자의 디지털 생활 안전을 위한 AI 통합 플랫폼으로, **음성 바코드 리더**, **보이스피싱 실시간 감지**, **키오스크 AI 도우미**, **보호자 대시보드** 4가지 핵심 기능을 제공합니다.
- Claude AI, Web Speech API, ZXing 바코드 인식 등 최신 AI 기술을 활용하여 고령자가 디지털 기기를 쉽고 안전하게 사용할 수 있도록 지원합니다.
- PWA(Progressive Web App) 기술로 설치 없이 모바일/PC에서 즉시 사용 가능하며, 오프라인에서도 일부 기능을 지원합니다.

## 최종목표 (2줄 이내)

- 고령자의 디지털 접근성을 향상시키고, 보이스피싱 등 디지털 범죄로부터 보호하여 **누구나 안전하고 편리한 디지털 생활**을 영위할 수 있도록 지원
- 보호자와 실시간 연동하여 가족 모두가 안심할 수 있는 **통합 AI 생활 안전 생태계** 구축

## 개발 내용 및 결과

### 1. 음성 바코드 리더 (들리는 바코드)
- **기능**: 바코드 스캔 시 식약처 공공데이터 API를 조회하여 제품명, 제조사, 원재료, 알레르기 유발 성분, 주의사항을 **음성(TTS)**으로 안내
- **기술**: ZXing 바코드 인식 라이브러리, Web Speech API (TTS), 식품의약품안전처 공공데이터 연동
- **결과**: 시각장애인 및 고령자가 제품 정보를 쉽게 확인할 수 있어 안전한 식품 소비 지원

### 2. 보이스피싱 실시간 감지 (AI 안심 가드)
- **기능**: 통화 내용을 실시간 분석하여 보이스피싱 패턴(기관 사칭, 개인정보 요구, 송금 요구 등)을 탐지하고 **즉시 경고**
- **기술**: Claude AI (자연어 처리), Web Speech API (STT), 패턴 매칭 알고리즘, 위험도 분석
- **결과**: 금융감독원/검찰 사칭, 대출 사기, 가족 사칭 등 10가지 유형의 피싱 시나리오 탐지 가능

### 3. AI 키오스크 도우미 (벗님)
- **기능**: 키오스크 화면을 인식하여 **단계별 음성 안내** 제공 (패스트푸드, 카페, 기차표, ATM, 병원 접수)
- **기술**: 화면 인식 AI, 시나리오 기반 가이드, 음성 합성
- **결과**: 키오스크 사용이 어려운 고령자도 독립적으로 주문/결제 가능

### 4. 보호자 대시보드
- **기능**: 연결된 고령자의 **실시간 안전 상태**, 활동 기록, 보이스피싱 차단 내역을 모니터링
- **기술**: 실시간 데이터 동기화, 위험 알림 시스템, 활동 통계 시각화
- **결과**: 보호자가 원격에서 가족의 안전을 확인하고 즉각 대응 가능

### 5. PWA 모바일 앱
- **기능**: 웹 브라우저에서 **홈 화면에 추가**하여 네이티브 앱처럼 사용, 오프라인 지원
- **기술**: next-pwa, Service Worker, Web App Manifest
- **결과**: 별도 앱 설치 없이 모바일에서 즉시 사용 가능

## 개발 과정에서 애로점

### 1. 바코드 인식 정확도 문제
- **문제**: 다양한 조명 환경과 바코드 상태에서 인식률이 낮은 경우 발생
- **해결**: ZXing 라이브러리의 TRY_HARDER 옵션 활성화, 다중 바코드 포맷 지원, 사용자에게 스캔 가이드 제공

### 2. 식약처 API 데이터 한계
- **문제**: 일부 제품이 공공데이터에 등록되지 않아 조회 실패
- **해결**: PocketBase 내부 데이터베이스를 백업으로 구축, 조회 실패 시 대체 데이터 제공

### 3. Next.js 16과 next-pwa 호환성
- **문제**: Next.js 16의 Turbopack과 webpack 설정 충돌
- **해결**: turbopack 설정 추가 및 --webpack 플래그 사용으로 빌드 환경 최적화

### 4. 보이스피싱 탐지 정확도
- **문제**: 정상적인 금융 상담과 피싱 전화 구분의 어려움
- **해결**: Claude AI의 맥락 분석 기능 활용, 복합 패턴 매칭으로 오탐지율 최소화
