 3. 기술 개요

SafeLife는 고령자의 디지털 생활 안전을 위한 AI 통합 플랫폼입니다.

 핵심 기능 4가지

1. 음성 바코드 리더 (들리는 바코드)
   - 바코드 스캔 시 제품 정보를 음성으로 안내
   - 식약처 공공데이터 API 연동

2. 보이스피싱 실시간 감지 (AI 안심 가드)
   - Claude AI 기반 통화 내용 분석
   - 10가지 피싱 유형 탐지

3. AI 키오스크 도우미 (벗님)
   - 키오스크 화면 인식 및 단계별 음성 안내
   - 패스트푸드, 카페, 기차표, ATM, 병원 접수 지원

4. 보호자 대시보드
   - 실시간 안전 상태 모니터링
   - 위험 알림 시스템

 기술 스택
- Frontend: Next.js 15, TypeScript, Tailwind CSS
- AI: Claude AI, Web Speech API (STT/TTS)
- 바코드: ZXing 라이브러리
- 배포: Vercel, PWA
